5. Forms

form groups inputs. label describes a field. input collects user data. button submits the form.

<form>
  <label for="name">Name:</label>
  <input type="text" id="name">
  <label for="email">Email:</label>
  <input type="email" id="email">
  <button>Submit</button>
</form>

6. Box Model

margin is the outer gap between elements. padding is the inner gap between content and border. border is the visible frame around the element.

.box { margin: 20px; padding: 16px; border: 3px solid #1a6fff; }

7. Typography

font-size sets text size. font-weight controls thickness. line-height sets spacing between lines for better readability.

.small { font-size:12px; font-weight:400; line-height:1.2 }
.large { font-size:28px; font-weight:700; line-height:1.8 }

Small — 12px · weight 400 · line-height 1.2

Medium — 18px · weight 600 · line-height 1.5

Large — 28px · 700

8. Colors & Background

color sets text colour. background-color fills the element. linear-gradient() creates smooth colour transitions.

.dark-box { color:#fff; background-color:#0a0c10 }
.grad     { background: linear-gradient(130deg,#1a6fff,#0a0c10) }
color: #fff · background-color: #0a0c10
linear-gradient(130deg, #1a6fff → #0a0c10)

10. Pseudo-classes

:hover applies styles when the mouse is over an element. :focus triggers when an input or button receives focus.

.btn:hover  { background:#0d4fcc; transform:scale(1.04) }
input:focus { border-color:#1a6fff; background:#f0f5ff }

11. DOM Manipulation

createElement() creates a new element in memory. appendChild() inserts it into the page — no reload needed.

addBtn.addEventListener('click', () => {
  const p = document.createElement('p');
  p.textContent = 'Item #' + (++n);
  container.appendChild(p);
});

12. Events & Style Toggle

classList.toggle() adds a class if missing, removes it if present. Combined with CSS variables it switches the entire page theme.

themeBtn.addEventListener('click', () => {
  document.body.classList.toggle('dark');
});
